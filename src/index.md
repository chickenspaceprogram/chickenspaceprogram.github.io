# athweb

You've reached the personal webpage of Athena Boose.

I'm a Physics and Computer Engineering student primarily interested in systems
programming, compilers, and programming language design. I'm also interested in
atomic physics and physical chemistry, but there's less of a focus on those
latter topics here.

If you want to contact me, check out the [contact](contact.html) page.

To see a few blog articles I've written about various topics, check out the
[blogs](/blogposts/index.html) page.

This site is statically generated with my own Unix-based toolchain. No
Javascript was used in its creation, it should work sensibly and be mostly
readable in [Lynx][lynx] or any other browser that doesn't support Javascript,
with the caveat that any math expressions might not display right on
browsers without MathML support.

Source code for this website is available [here][athweb] under a CC BY 4.0
license. Source code for most of my other projects is available on
[Codeberg][aths-cb]; this project hasn't been moved there because I haven't
bothered to migrate this site from Github Pages' free hosting to my own server.

Building this site requires [Pandoc][pandoc] as well as my own preprocessor,
[hmacro][hmacro]. Eventually I'll replace these dependencies, as `hmacro` is
not a good preprocessor and Pandoc is slower than I'd like.

If you know me personally and want to host your own site or receive email on a
subdomain of this one, contact me and I'll get that set up for you.
